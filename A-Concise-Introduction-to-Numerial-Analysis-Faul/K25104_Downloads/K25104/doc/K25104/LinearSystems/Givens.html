Givens

PURPOSE

Applies one Givens rotation to place a zero in the (i,j)th entry of A.

SYNOPSIS

function [ B,G ] = Givens( A,i,j )

DESCRIPTION 

 Applies one Givens rotation to place a zero in the (i,j)th entry of A.
 Input arguments:
   A, matrix
   i, row index
   j, column index, j<i
 Output arguments:
   G, Givens rotation
   B, B = G*A
